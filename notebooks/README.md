QF206 Notebooks
================

This folder contains interactive teaching and interpretability notebooks
for the QF206 quant trading project.

- `demo_six_stock_walkthrough.ipynb`:
  Step-by-step walkthrough on the fixed six-stock subset
  (`AAPL`, `MSFT`, `NVDA`, `JPM`, `XOM`, `UNH`) covering:
  - Loading the monthly panel
  - Inspecting features and labels
  - Visualising forward 1M returns and simple factor relationships
