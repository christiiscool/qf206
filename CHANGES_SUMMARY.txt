PART 5: ADVANCED FEATURES (2 IMPLEMENTED) ✅
================================================================================
OPTION A: ALLOCATION SMOOTHING ✅
  Formula: w_t = γ*w_target + (1-γ)*w_{t-1}
  Config: allocation_smoothing = 0.7 (default)
  Effect: Reduces turnover, smoother transitions
  
  γ = 1.0 → No smoothing
  γ = 0.7 → 70% new, 30% old (default)
  γ = 0.5 → Equal weighting
  
OPTION B: VOLATILITY TARGETING ✅
  Formula: Scale returns to maintain target vol
  Config: enable_vol_targeting = False (default)
          target_vol = 0.12 (12% annualized)
          vol_lookback = 12 (months)
  Effect: Maintains consistent risk exposure
  Safety: Scaling capped at [0.5, 2.0]
